🚀 Free Deployment Guide

1. GitHub Pages (Free): GitHub पर एक repository बनाएं, यह file upload करें। Settings → Pages → branch select करें → Save। 2 मिनट में आपकी site live हो जाएगी (yourname.github.io/repo-name)।

2. Netlify (Free): netlify.com पर जाएं → "Add new site" → "Deploy manually" → file को drag-drop करें। तुरंत live link मिल जाएगा।

3. Vercel (Free): vercel.com पर sign up करें, GitHub repo import करें, Deploy दबाएं।
